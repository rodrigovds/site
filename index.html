Formatação completa com os sistemas abaixo:

  • Windows 10 Lite version;

  • Windows 10 normal version;

  • Windows 7 Profissional;

A escolha vai depender do sistema que for liberado pelo fabricante do seu computador.


Programas instalados para utilização:

  • Pacote Office 2010 (Word, Excel, Power Point, Outlook);

  • Pacote de codec para Windows Media Player (original do Windows);

  • Programa para gravação de CDs e DVDs caso tenha o Hardware para tal função;

  • Google Chrome navegador internet

  • Foxit PDF para leitura de arquivos PDFs;

  • Photofiltre para visualizar, organizar e editar arquivos de imagem;

  • Winrar para compactar e descompactar vários arquivos;
